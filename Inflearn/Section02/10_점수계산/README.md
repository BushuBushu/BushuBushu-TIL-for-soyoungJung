## 설계(풀이 방식)
- scores 리스트 값은 기본 0이 된다. 
- 해당 인덱스 앞의 result가 0이라면, 또한 해당 인덱스의 result가 1이라면,  해당 인덱스의 score는 1이 된다.
- 해당 인덱스 앞의 result가 1이라면, 또한 해당 인덱스의 result가 1이라면, 해당 인덱스의 score는 이전 인덱스의 score에 1을 다시 더한 값이 된다.

### main

1. 결과를 저장하는 results 리스트

2. 점수를 저장하는 scores 리스트

3. 1.과 2.앞에 모두 0을 추가한다. --> 이전 인덱스에 모두 접근할 수 있도록.

## 강의 해설
- result의 값이 1일 경우, 가중치에 1을 더하고, 그 가중치를 sum에 더하여 최종 sum을 출력하였다.
- result의 값이 0인 경우, 가중치를 0으로 초기화 시켰다.

## 배운 개념 및 깨달은 점
굉장히 단순하고, 필요있는 변수인 `가중치`를 설정한 것이 인상깊었다.
