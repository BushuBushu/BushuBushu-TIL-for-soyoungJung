## 배운 개념 및 깨달은 점
- 리스트가 비어있으면 `False`이고, 아니라면 `True`이다. 다음과 같이 사용할 수 있다.
```
p = [1, 2, 7]
#p가 비워질때까지 pop하는 것
while p:
    p.pop()
```
- deque 자료구조
list는 pop할 경우 하나씩 당겨지는데, deque는 삭제되면 포인터가 다음거로 넘어간다. 당겨지지 않는다.
