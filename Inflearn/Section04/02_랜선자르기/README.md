## 설계(풀이 방식)
1. maxx
랜선의 최대 길이. 따라서 구하려는 랜선의 길이는 maxx와 같거나 작다. why? 주어진 랜선을 모두 이어붙여 N만큼 나눈 길이이기 때문.

2. summ
주어진 랜선 길이의 값을 maxx로 나눈 몫의 합. for문을 돌며 구해진 몫을 더한 것이 N이 될때 반복문 종료.
```
summ = 0
while summ != N:
    summ = 0
    for i in l:
        #리스트 각 값을 maxx로 나누면 몫이 나옴. summ은 몫의 합
        summ += i // maxx
    print('sum',summ)
    if summ != N:
        maxx -= 1
```
