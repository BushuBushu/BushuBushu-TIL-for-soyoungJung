## 배운 개념 및 깨달은 점
1. pop()
- pop()을 사용하여 리스트에서 어떤 값을 지울 수 있을 뿐만 아니라 pop()의 return값, 즉 지워진 값을 활용할 수도 있다.
2. insert()
- insert : 리스트.index(입력할index, 값)
```
>>> a = [1, 2, 3]
>>> a.insert(1, 5)
>>> a
[1, 5, 2, 3]
```
